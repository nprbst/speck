# Speck Installation Process

This demo shows the complete Speck installation process using Claude Code's plugin marketplace.

## Steps shown:

1. Type `/plugin` to open the Claude Code plugin manager
2. Add the speck-market marketplace
3. Install the Speck plugin
4. Verify installation with `/speck.env`

After installation, you can immediately start using Speck commands like `/speck.specify`, `/speck.plan`, and `/speck.tasks` to build specification-driven features.
