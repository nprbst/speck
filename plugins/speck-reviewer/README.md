# Speck Reviewer

AI-powered PR review with Speck-aware context.

## Quick Start

1. Install: `/speck-reviewer:init`
2. Review a PR: `/speck-reviewer:review [pr-number]`

## Features

- **Cluster Analysis**: Groups changed files semantically
- **Speck Integration**: References spec requirements in reviews
- **Comment Management**: Stage, refine, batch-post comments
- **Self-Review Mode**: Review your own PRs (posts as issue comments)

## Documentation

See full documentation at the [Speck website](https://speck.dev/plugins/speck-reviewer).

## Requirements

- Bun runtime
- GitHub CLI (`gh auth login`)
